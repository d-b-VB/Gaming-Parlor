# Open Design Questions

These are intentionally not blockers for the first prototype.

1. Should Club odds be shown as simple multipliers, implied probabilities, or both?
2. What empirical/probabilistic model should eventually replace the first simple target estimator?
3. Should Heart safety use the estimated 50% target, a softer target, or an independent risk model?
4. Should pause breaks consume a charge, trigger automatically, or be activated manually?
5. Should category-choice and category-rearrangement upgrades happen before every round or only occasionally?
6. Should exact-palette color categories eventually be shown as text labels, color swatches, or both?
7. Should ambiguous boards be rejected strictly, or should some ambiguity be allowed at higher difficulty?
8. Should flags be common in normal play or reserved for geography/color-heavy rounds?
